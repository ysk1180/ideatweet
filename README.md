# README

## サービス名
アイデアTweet  
https://ideatweet.herokuapp.com/

## 概要
ランダムに表示される2つのキーワードからアイデアを発想し、画像付きでツイートできるサービス  
LINE botバージョンもあります（サービスのサイト下部のお友達追加リンクより）

## 技術チュートリアル
